# Constants

There is a number of constants, that you can use in CommandLang code. To use them, add this to any raw command `${<constant name>}`.

## List

| Name               | Value                           |
|--------------------|---------------------------------|
| `RAM_ENTITY`       | Current ram entity identifier   |
| `RAM_SELECTOR`     | Selector for current ram entity |
| `this`             | Selector for current ram entity |
| `CC_BLACK`         | `§0`                            |
| `CC_DARK_BLUE`     | `§1`                            |
| `CC_DARK_GREEN`    | `§2`                            |
| `CC_DARK_AQUA`     | `§3`                            |
| `CC_DARK_RED`      | `§4`                            |
| `CC_DARK_PURPLE`   | `§5`                            |
| `CC_GOLD`          | `§6`                            |
| `CC_GRAY`          | `§7`                            |
| `CC_DARK_GRAY`     | `§8`                            |
| `CC_BLUE`          | `§9`                            |
| `CC_GREEN`         | `§a`                            |
| `CC_AQUA`          | `§b`                            |
| `CC_RED`           | `§c`                            |
| `CC_LIGHT_PURPLE`  | `§d`                            |
| `CC_YELLOW`        | `§e`                            |
| `CC_WHITE`         | `§f`                            |
| `CC_OBFUSCATED`    | `§k`                            |
| `CC_BOLD`          | `§l`                            |
| `CC_STRIKETHROUGH` | `§m`                            |
| `CC_UNDERLINE`     | `§n`                            |
| `CC_ITALIC`        | `§o`                            |
| `CC_RESET`         | `§r`                            |
